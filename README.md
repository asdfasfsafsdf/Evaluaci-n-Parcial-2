# Documentación del Desarrollo del Sistema de Proyectos

## Herramientas Usadas
* **React (Create React App):** Framework principal para el desarrollo de la interfaz de usuario y la lógica de componentes.
* **Node.js & npm:** Entorno de ejecución y gestor de paquetes para librerías y scripts de construcción.
* **Firebase CLI:** Herramienta de línea de comandos para la autenticación, inicialización y despliegue del hosting.
* **Firebase Hosting:** Servicio de alojamiento en la nube para archivos estáticos y aplicaciones SPA.
* **Inteligencia Artificial (LLM):** Asistente para generación de código, diseño de estilos y resolución de errores de despliegue.

## Dónde fue útil la IA
1.  **Aceleración del Desarrollo:** Ayudo en el proceso del desarrollo del codigo.

## Reflexión sobre la utilidad
La IA mejoro flujo de trabajo que normalmente requeriría consultar documentación técnica. Evitando el hecho de tener que investigar en diferentes páginas, ahora puedes preguntar cualquier cosa a la IA y tener la respuesta en una sola ventana. Ademas que en problemas mucho más tecnicos puede ser más rapido al encontrar el error.
